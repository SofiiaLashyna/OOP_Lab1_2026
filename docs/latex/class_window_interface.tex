\doxysubsection{Detailed Description}
The main QWidget class that serves as the entry point of the UI. 

This class holds the main stacked widget, the start screen, and the \doxylink{class_galaxy_view}{Galaxy\+View}. It is responsible for loading the initial JSON data and triggering the galaxy generation.
