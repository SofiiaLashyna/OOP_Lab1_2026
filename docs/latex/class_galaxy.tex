class Galaxy$<$ Graph\+Type $>$}
Manages all celestial objects and their relationships (graph) in a galaxy. 

This class acts as the top-\/level container, owning all \doxylink{class_celestial_object}{Celestial\+Object} pointers (\doxylink{class_star_system}{Star\+System}, \doxylink{class_nebula}{Nebula}, etc.) and managing the graph that connects them.
