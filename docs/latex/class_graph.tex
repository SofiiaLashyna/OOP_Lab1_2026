\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Graph$<$ T $>$}
An abstract base class defining the interface for a graph structure. 

This class provides the core storage (vertices, edges) and a common interface that concrete classes (\doxylink{class_graph_list}{Graph\+List}, \doxylink{class_graph_matrix}{Graph\+Matrix}) must implement.
