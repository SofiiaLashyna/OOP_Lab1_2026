\doxysubsection{Detailed Description}
The main UI widget, inheriting from QWidget. 

This class manages the {\ttfamily \doxylink{class_graph_widget}{Graph\+Widget}} display, floating parameter windows, galaxy generation, and user interaction (clicks, edits, zooming).
