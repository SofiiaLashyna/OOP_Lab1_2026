\doxysubsection{Detailed Description}
Contains implementations of classical graph algorithms (BFS, DFS, Dijkstra) implemented as a templated utility class. 

This file also defines an abstract interface \doxylink{class_graph_algorithm}{Graph\+Algorithm} for implementing the Strategy pattern when selecting graph algorithms at runtime.
