\subsubsection*{template$<$typename T$>$\newline
class Dijkstra\+List\+Algorithm$<$ T $>$}
Strategy class that runs Dijkstra\textquotesingle{}s shortest-\/path algorithm on a graph represented by an adjacency list. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & The data stored in graph vertices.\\
\hline
\end{DoxyTemplParams}
The algorithm computes the minimum path cost from {\ttfamily start\+Id} to {\ttfamily end\+Id} using a priority queue. Works only with non-\/negative weights.

The result (shortest distance) is returned and also printed to std\+::cout. \begin{Desc}
