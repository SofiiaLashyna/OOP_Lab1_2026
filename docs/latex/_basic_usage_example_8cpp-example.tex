\doxysection{Basic\+Usage\+Example.\+cpp}
\hypertarget{_basic_usage_example_8cpp-example}{}\label{_basic_usage_example_8cpp-example}This is an example of how to\+:
\begin{DoxyEnumerate}
\item Create Stars, Planets, and Nebulae.
\item Group them into Star\+Systems.
\item Add systems and nebulae to a polymorphic \doxylink{class_galaxy}{Galaxy}.
\item Connect objects and run algorithms (Dijkstra, BFS, DFS) on the graph.
