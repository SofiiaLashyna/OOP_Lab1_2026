﻿# Project1 Galaxy Simulation is a C++ programming educational project that models the structure of a galaxy using graphs. Each astronomical object (star, star system, planet, etc.) is represented as a vertex of a graph, and the connections between them are represented as edges with weights (e.g., distance between systems).

The project demonstrates:

  • working with template classes;
  • using an adjacency list and an adjacency matrix to represent graphs;
  • implementing graph traversal algorithms (BFS, DFS) and pathfinding (Dijkstra);
  • an example of object-oriented modeling of astronomical objects;
  • unit testing to verify the correctness of the main components;
  • graphical construction of a graph of elements.

The project contains a set of unit tests that automatically verify:

  • creation and initialization of graphs;
  • adding/removing vertices and edges;
  • correctness of traversal algorithms;
  • correctness of connections in Galaxy.
